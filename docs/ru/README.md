# HapticTrace

`HapticTrace` — desktop-инструмент для записи и анализа событий с вибро откликом(haptic). Он объединяет сенсорные данные из `phyphox`, захват экрана iPhone/iPad и инструменты пост-анализа. После записи можно сопоставить видео с измеренным сигналом, скорректировать offset и сохранить результат для повторного разбора.

**Платформа:** только macOS

## Что умеет

- захватывает сенсорные данные через `phyphox`
- показывает live preview и записывает экран iPhone/iPad
- поддерживает режимы записи: `Sensors + Video`, `Sensors Only`, `Video Only`
- отображает сигнал и спектрограмму
- воспроизводит записанное видео на общем timeline
- поддерживает автоматическую и ручную подстройку offset
- сохраняет и загружает сессии в `.zip`
- сравнивает несколько сессий

## Что нужно для работы

- macOS
- Python 3
- устройство с `phyphox`, доступное по сети
- iPhone или iPad, доступный в macOS как источник screen capture
- доступ к интернету при первом запуске из исходников

## Основной сценарий

Основной режим работы — синхронная запись сенсорных данных и экранного контекста с последующим анализом.

В типовой конфигурации используются два мобильных устройства:

- **Устройство 1(телеон)** — сенсорный модуль с `phyphox`  
  Используется для захвата данных `accelerometer` и `gyroscope`.
- **Устройство 2(телеон)** — целевое iPhone/iPad  
  На нем запускается тестируемое приложение, воспроизводятся хаптики и захватывается экран.
- **Mac** — хост с `HapticTrace`  
  Подключается к сенсорному устройству по сети и к iPhone/iPad через usb к macOS screen capture.

### Подготовка

1. На сенсорном устройстве откройте `phyphox`.
2. Создайте или загрузите эксперимент, в котором включены `accelerometer` и `gyroscope`.
3. Включите в `phyphox` remote access и получите URL эксперимента.
4. Подключите iPhone/iPad к Mac через usb как источник захвата экрана.
5. Жестко зафиксируйте оба устройства относительно друг друга(например будербродом):
   - iPhone/iPad — экраном вверх
   - сенсорное устройство — экраном вниз

Во время записи устройства не должны смещаться друг относительно друга. Любое паразитное движение напрямую влияет на качество измерения.

### Workflow

1. Запустите `HapticTrace` на Mac.
2. Подключите сенсорное устройство по URL `phyphox`.
3. Подключите iPhone/iPad как источник видео.
4. Запустите запись(кнопка Play в главном окне).
5. Воспроизведите хаптики в тестируемом приложении(на iPhone/iPad).
6. Остановите запись.
7. Проанализируйте сигнал и синхронизируйте его с действиями на экране по общему timeline.

### Дополнительные режимы

Помимо основного синхронного сценария, `HapticTrace` поддерживает и раздельные режимы записи:

- **`Sensors Only`** — только захват сенсорных данных без видео
- **`Video Only`** — только запись экрана без сенсоров

## Запуск

### Из исходников

#### Из консоли

```bash
./run_app.sh --url http://<device_local_ip>:8080
```

При первом запуске скрипт создаст `.venv`, обновит `pip` и установит runtime-зависимости автоматически.

#### Из Finder

```bash
run_app.command
```

### Release

Откройте `HapticTrace.app` из подготовленной release-сборки.
